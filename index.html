  • ダーク / ライト / システム追従の 3 テーマ、17 言語 UI（x64 / ARM64）
  • Native AOT で .NET 不要、Velopack 自動更新（Cloudflare R2 lhamiel.nephilim.jp）

Ferry

QR コードでペアリングしPC 間のファイルを P2P 転送するデスクトップアプリ。

  • スマホを「橋渡し」にした QR ペアリング（Firebase Hosting Bridge ページ＋カメラスキャン）、ペア情報はローカル peers.json に保存して Firebase 即削除
  • 接続経路 3 階層フォールバック: TCP 直接 → UDP ホールパンチ（STUN）→ WebSocket リレー（Cloudflare Workers + Durable Objects Hibernation）
  • 接続経路を 🟢 LAN 直接 / 🟡 P2P / 🔴 リレー で UI 可視化
  • 長さプレフィクス付きバイナリプロトコル、SHA-256 検証、64KB チャンク、転送レジューム対応
  • Avalonia UI 12 + .NET 10 Native AOT（Win x64 / ARM64・macOS arm64・Linux x64 / ARM64）
  • Velopack 自動更新（Cloudflare R2 ferry.nephilim.jp）
