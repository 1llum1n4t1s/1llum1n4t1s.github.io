SuperLightLogger
『log4netやNLogを`using`を1行書き換えるだけで卒業できる』をコンセプトにした超軽量な互換シム。`using log4net;`を`using SuperLightLogger;`に置換＋`Program.cs`で`LogManager.Configure(...)`を1回呼ぶだけで、既存の`log.Info()`/`log.DebugFormat()`/`log.Error(msg, ex)`などをコード無改造でC#標準`Microsoft.Extensions.Logging`へ移行可能（所要数分）。Serilog / OpenTelemetry / Application Insights / Datadog / Seq / Loki等のシンクが設定だけで使えるようになり、構造化ログ（`log.InfoStructured("ユーザー {UserId} ...", ...)`）へ段階的に移行可能、ASP.NET Core / Generic Host / DIともシームレス統合。NLog互換の`File Target`（日付/サイズベースローテーション、アーカイブ）を追加依存ゼロで内蔵、`net8.0` / `net10.0`で`IsAotCompatible=true`対応、2.6MB単一ネイティブEXE内での動作を実機検証済み。
