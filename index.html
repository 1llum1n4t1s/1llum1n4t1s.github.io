  • ロック中ファイルの自動一時コピー圧縮
  • 暗号化 ZIP / 7z / RAR のパスワード入力ダイアログ
  • 二重ネスト防止 ＆ Windows 風サムネ付き衝突ダイアログ
  • 事前＆処理中のディスク容量チェック、破損アーカイブ救出
  • ダーク / ライト / システム追従の 3 テーマ、17 言語 UI
  • Native AOT で .NET 不要、Velopack 差分自動更新

Komorebi

C# / .NET 10 + Avalonia UI 12 で構築されたクロスプラットフォーム Git GUI クライアント（SourceGit フォーク）。

  • 初回セットアップウィザード、OS ロケール自動検出＋フォント最適化
  • Squash / Fixup 後の --force-with-lease 強制プッシュ
  • ブランチ作成後の自動プッシュ、リポジトリ自動スキャン
  • Staged / Unstaged 配置入れ替えで直感的な UI
  • SSH 鍵 3 段フォールバック（リモート個別 → グローバル → ssh-agent）
  • AWS CodeCommit の HTTPS / SSH / git-remote-codecommit 3 形式対応
  • OneDark / White テーマ追加、17 言語翻訳率 100%
  • Velopack 自動アップデート
MIT Avalonia 12 Win / Mac / Linux 17言語

  • 起動時自動開始、最小化でタスクトレイ格納

  • 11 プリセット: オフ / 3 時間 / 12 時間 / 1 日 / 3 日 / 1 週間 / 1 ヶ月 / 3 ヶ月 / 半年 / 1 年 / 3 年

  • using System.IO; を using Alphaleonis.Win32.Filesystem; に置換するだけでドロップイン互換
  • COM Interop を .NET 10 / NativeAOT 対応のため全面刷新
